# refinery-planning-benchmark
This repository provides a Python implementation of a production planning benchmark based on real-world refinery-petrochemical complex data. The original repository is
https://github.com/EMRPS/refinery-planning-benchmark. In the original repository, computational results were obtained from global optimization solvers ANTIGONE and BARON. However, since ANTIGONE and BARON require paid licenses, this repository solves the problem using free solvers.
